This program is designed to run on the GCC compiler obtained via Windows MinGW.  You can also execute the build .exe file.

This program resizes a 1-D array of “8-bit pixels” based on the specified source and destination lengths. During our meeting, I conceptualized the implementation but couldn't finish the code due to time constraints. Here is the complete, executable code. It includes a demo that shows how a stretched 'image' can be shrunk back to its original size.
